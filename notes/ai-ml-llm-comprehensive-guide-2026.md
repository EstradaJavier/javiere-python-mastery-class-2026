- **Artificial Intelligence (AI)** â€” The grand umbrella: machines performing tasks requiring human-like intelligence (problem-solving, perception, decision-making). Includes rule-based systems and learning-based ones.

- **Machine Learning (ML)** â€” Core method inside AI: algorithms learn patterns from data to make predictions/decisions without hard-coded rules.

- **Deep Learning (DL)** â€” Advanced ML using deep neural networks (many layers). Excels at unstructured data like images/text. Powers most modern AI.
